# Project Images

Place your project images in this folder with the following names:

- `website-audit-tool.jpg` - Image for Website Audit Tool project
- `aadhaar-awareness.jpg` - Image for Aadhaar Awareness Platform project
- `health-monitoring.jpg` - Image for Smart Community Health Monitoring project
- `virtual-mouse.jpg` - Image for Virtual Mouse Using AI project
- `ai-translator.jpg` - Image for AI Translator project
- `assistive-wearable.jpg` - Image for AI-Driven Assistive Wearable project

## Image Recommendations:
- Recommended size: 800x600 pixels or similar aspect ratio
- Format: JPG, PNG, or WebP
- File size: Keep under 500KB for faster loading
- If an image is not found, a placeholder icon will be displayed automatically
